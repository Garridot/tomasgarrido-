## Installation and Usage

1. Clone this repository: 
```bash
git clone https://github.com/Garridot/tomasgarrido-.git
```
2. Install dependencies:

```bash  
pip install -r requirements.txt

```
3. Run the application:

```bash  
python run.py
```
